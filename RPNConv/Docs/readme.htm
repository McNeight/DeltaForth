Purpose

The purpose of this program is to translate the algebraic form of an expression to the RPN form (Reverse Polish Notation) and further more, issue a set of Forth primitives that - based on the initial values on stack - perform the calculation of the expression. The RPN form will be generated for any degree of complexity of the input expression, whereas the Forth form will be generated for low and medium complexity expressions. This is not a limitation of the program itself but of the way Forth handles the stack.

 

System Requirements

The program can run on any system supporting Java. It was compiled using Sun's Java Developer Kit 1.2.2. There are no other special requirements.

 

Syntax

To use the program, you need to supply two parameters, of which the first is compulsory:

java RPNConv "Expression in Algebraic Form" ["Initial Stack Layout"]

"Expression in Algebraic Form" is what the name says. This is the expression you provide in algebraic form:

(A+B)*(A-B)+15*C         or         X+(Y*(X-19))*Z

The program will transform these into their RPN equivalents, as follows:

A B + A B - * 15 C * +             X Y X 19 - * Z * +

If you specify the initial stack layout, the program will do its best to create the sequence of Forth primitives that calculate the expression. If the expression is too complex or not all parameters are specified, an error is issued. Note that the parameter order can affect the efficiency of the sequence. You may want to try different combinations unless a certain combination is required. Duplicate parameters will be ignored. Here are some examples of stack layout:

AB    ACB    YZX

 

Stack Notation

When specifying the initial stack layout, the first parameter you specify is placed at the bottom of the stack:
