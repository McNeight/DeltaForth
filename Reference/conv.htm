String Conversion

Forth uses a very powerful string conversion scheme and it is very different of what we can find in other systems. Below is a description of the primitives used in converting an integer from the stack to a string. For an example of using string conversion primitives, please see the example files.

Word Description / Action
<# ( n --- ) Starts conversion of the integer on the top of stack.
# Converts one digit.
#S Converts all digits.
HOLD ( c --- ) Places on the output string the character c.
SIGN Places a "-" in the output string if the number is negative.
#> ( --- addr n) Ends conversion. Leaves address and length of string on the stack.
