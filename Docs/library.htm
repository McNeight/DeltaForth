JAVA Libraries

Version 0.8 of the DELTA Forth system introduced the concept of external libraries. Although well-known in other programming environments, Forth did not have its own library system, up until recently. Using external JAVA libraries, DELTA Forth can virtually do anything.

To create a JAVA library, follow these steps:

1. Create a 4th file in which you declare the words of the library as externals.

Example:

EXTERNAL FOPEN IO    \ Declares the external word FOPEN in library IO

2. Write the corresponding Java functions.

Example:

class IO {

public void FOPEN(int Memory[], int P[]) {

// write your code here

}

}
The Memory[] array gives you access to the DELTA Forth addressable space. P[0] contains the SP system variable and P[1] contains the RP system variable. If you modify these parameters, the changes will be transmitted to the Forth engine accordingly.

3. Compile the Java code and place the generated CLASS file where the interpreter can find it (usually own directory).

4. Write your Forth code, including the declarations with a LOAD directive (see example for details).

IMPORTANT!

The names of Java classes and member functions are required to be written in uppercase.
