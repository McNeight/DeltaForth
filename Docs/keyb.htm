Keyboard Operations

Word Description / Action
KEY ( --- c ) Places on the stack the ASCII code of the key pressed
EXPECT ( addr n --- ) Reads n characters and places their codes starting from address addr
?TERMINAL ( --- b ) Returns a boolean value specifying whether a key has been pressed
QUERY ( --- ) Awaits maximum 80 characters to be typed and places them at the area pointed by TIB
