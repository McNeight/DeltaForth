System Variables

Word Description / Action
PAD Points to a 64 cells wide work area
S0 Parameter stack origin
R0 Returns stack origin
STATE Returns the state of the DELTA Forth system. The returned value is always 0 (compiler state). Included for compatibility with other Forth systems.
BASE Returns the current working radix. Possible values are 2, 8, 10 and 16.
TIB Points to temporary buffer used in I/O operations
